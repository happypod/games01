# 제품 명세

## 1. 제품 가정

- 장르: 싱글 플레이 판타지 방치형 RPG
- 플랫폼: 모바일·데스크톱 반응형 웹
- 저장: 브라우저 `localStorage`
- 목표 경험: 영웅과 동료의 자동 전투를 지켜보고, 골드를 성장에 재투자하고, 보스 관문을 넘은 뒤 환생으로 다음 원정을 가속한다.
- 첫 수직 조각의 성공 기준: 30~45분 안에 첫 환생 관문을 경험한다.

## 2. 핵심 루프

```text
영웅 자동 전투 → 골드·경험치 → 장비·스킬 성장 → 다음 스테이지
       ↑                  동료 영입·훈련          ↓
       └──────── 이전 스테이지 파밍 ← 보스에서 패배 ┘

30스테이지 도달 → 환생 → 영구 보너스 → 더 빠른 재진행
```

- 5~30초: 적 처치와 보상 피드백
- 2~5분: 강화 선택과 보스 관문
- 30~45분: 첫 환생 판단
- 수 시간: 오프라인 결과 회수와 최고 기록 갱신

## 3. 현재 MVP 범위

### 포함

- 영웅 1명, 편성 동료 1명, 한 번에 적 1마리
- 첫 보스 승리 뒤 고정 동료 `불씨 여우 루미` 영입·훈련·자동 협공
- 자동 기본 공격과 자동 스킬
- 최대 300스테이지를 생성하는 데이터 기반 적 공식, 10스테이지마다 보스
- 골드, 경험치, 스킬 포인트, 불씨 정수
- 무기·갑옷·부적 강화
- 스테이지 진행·선택·패배 복귀
- 환생과 영구 공격력·체력 보너스
- 저장 버전, 손상 데이터 방어, 자동 저장
- JSON 파일 저장 백업·검증 후 복원
- 최대 8시간 오프라인 시뮬레이션

### 명시적 비범위

- 로그인, 서버 권위 시간, 클라우드 저장
- 결제, 광고, 유료 재화, 가챠
- 랜덤 장비와 인벤토리
- 둘 이상의 동료 동시 편성, 여러 영웅, 동료 장비·가챠·사망
- PvP, 친구, 길드와 서버 대전
- 퀘스트, 출석, 시즌, 라이브 이벤트

비범위 기능은 핵심 루프 지표가 안정된 뒤 별도 티켓으로만 진입한다.

## 4. 전투·성장 규칙

- 전투 라운드: 1초
- 플레이어가 먼저 공격하고, 적이 생존하면 반격한다.
- 처치 시 골드·경험치를 한 번 지급하고 다음 스테이지로 이동한다.
- 처치 시 최대 체력의 20%를 회복한다.
- 패배 시 한 스테이지 내려가 최대 체력으로 복귀한다. 최고 기록은 유지한다.
- `화염 강타`는 준비 상태에서 자동 시전되고 5초 쿨다운을 갖는다.
- 모든 플레이어 공격은 저장된 `xorshift32-v1` 순서에서 1회 판정하며, 15% 확률로 1.75배 치명타가 된다.
- 화염 강타와 치명타 배율을 모두 곱한 뒤 최종 피해를 한 번만 반올림한다.
- 첫 보스를 격파해 스테이지 11을 열면 불씨 여우 루미를 무료로 한 번 영입할 수 있다.
- 동료는 3초 cooldown이 준비됐고 영웅 공격 뒤 적이 생존했을 때만 협공한다. 동료 공격은 RNG·치명타·화염 강타를 사용하지 않는다.
- 영웅 또는 동료가 적을 처치해도 보상과 다음 스테이지 처리는 공통 분기에서 정확히 한 번만 실행된다.
- 동료 영입은 환생 뒤 유지하지만 원정 훈련은 랭크 1로 초기화한다.
- 레벨업마다 스킬 포인트 1점을 받는다.

주요 성장식은 코드의 단일 출처를 따른다.

```text
적 기본 체력   = 34 × 1.15^(stage - 1) × bossMultiplier
첫 환생 보정   = ratio^(stage - 1), stage ≤ 30
               = ratio^(29 × (60 - stage) / 30), 30 < stage < 60
               = 1, stage ≥ 60
ratio          = 1.188 / 1.15
적 공격력     = 4 × 1.105^(stage - 1) × bossMultiplier
강화 비용     = baseCost × costGrowth^currentLevel
정수 획득     = floor((highestStage / 10)^1.5), 30스테이지부터
정수 영구 효과 = 정수 1개당 공격력·체력 +4.2%
치명타 피해     = round(공격력 × 화염 강타 배율 × 1.75), RNG roll < 0.15
동료 협공 피해  = round(공격력 × (0.20 + 0.05 × (rank - 1)))
동료 훈련 비용  = round(100 × 1.8^(currentRank - 1))
```

첫 환생 보정은 31~59스테이지에서 기하급수적으로 감소해 60스테이지부터 1이 된다. 따라서 첫 환생 목표만 조정하고 장기 1.15 체력 곡선은 유지한다.

누적 자원·경험치·통계·전투 카운터와 전투 보고, 1회 피해·보상은 `Number.MAX_SAFE_INTEGER`에서 포화한다. 상한에 도달해도 다음 전투와 저장은 안전한 정수 계약을 유지한다.

모든 반올림, 최소값, 최대값은 [formulas.ts](../src/game/formulas.ts)와 [content.ts](../src/game/content.ts)를 기준으로 한다.

## 5. 오프라인 진행

- `현재 시각 - 마지막 저장 시각`을 `0~8시간`으로 제한한다.
- 온라인과 동일한 결정론적 전투 엔진을 사용한다.
- 시계가 역행하면 경과 시간을 0으로 취급한다.
- 적용 직후 기준 시각을 현재 시각으로 옮겨 동일 구간 중복 수령을 막는다.
- 로컬 전용 버전은 기기 시계를 신뢰한다. 경쟁·결제가 추가되면 서버 권위 시간이 선행되어야 한다.

## 6. 핵심 수용 시나리오

1. 신규 사용자는 진입 즉시 자동 전투 상태를 확인한다.
2. 적 처치 보상은 정확히 한 번만 지급된다.
3. 구매 성공 시 비용 차감과 강화가 원자적으로 일어난다.
4. 구매 불가 상태는 버튼 비활성화와 명확한 사유로 표현된다.
5. 패배해도 최고 기록은 사라지지 않고 이전 구간 파밍을 계속한다.
6. 새로고침 후 저장된 진행과 전투 상태가 복구된다.
7. 1시간과 8시간 초과 오프라인 진행이 정해진 상한 안에서 계산된다.
8. 손상 저장은 앱을 중단시키지 않고 새 게임으로 복구된다.
9. 환생 후 골드·레벨·장비는 초기화되고 정수·평생 통계는 유지된다.
10. 장시간 계산 뒤에도 HP·재화·스테이지에 `NaN`, `Infinity`, 음수가 없다.
11. 첫 보스 뒤 동료를 영입·훈련하면 3초 협공이 전투와 오프라인 결과에 나타나고 reload·환생 뒤 계약대로 복구된다.
